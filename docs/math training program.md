# Kesari Mathematics Expert Training Program

A structured 16-week curriculum to develop strong mathematical reasoning, problem-solving skills, and deep conceptual understanding from foundational arithmetic through advanced undergraduate mathematics.

---

## Program Overview

| Phase | Duration | Focus Area |
|-------|----------|------------|
| **Phase 1** | Weeks 1-4 | Foundations & Pre-Calculus |
| **Phase 2** | Weeks 5-8 | Differential Calculus & Linear Algebra |
| **Phase 3** | Weeks 9-12 | Integral Calculus & Multivariable Math |
| **Phase 4** | Weeks 13-16 | Advanced Topics: Analysis, Abstract Algebra |

**Study Schedule**: 5-6 days per week, 60-90 minutes daily
**Prerequisite**: Basic number sense (arithmetic operations)

---

## Phase 1: Foundations & Pre-Calculus (Weeks 1-4)

### Daily Schedule

| Day | Topic | Time | Activities |
|-----|-------|------|------------|
| **Day 1** | Arithmetic Review | 60 min | Operations, integer properties, divisibility rules |
| **Day 2** | Fractions & Decimals | 60 min | Conversions, operations, word problems |
| **Day 3** | Basic Algebra | 75 min | Expressions, equations, factoring |
| **Day 4** | Exponents & Radicals | 75 min | Laws, simplification, rationalizing |
| **Day 5** | Review & Practice | 90 min | Mixed problem set, self-assessment |
| **Day 6** | Assessment | 60 min | Quiz + concept review |

### Topics Covered

1. **Number Theory Basics**
   - Integers, rationals, irrationals, real numbers
   - Prime factorization, GCD, LCM
   - Modular arithmetic foundations

2. **Algebraic Foundations**
   - Polynomial expressions and operations
   - Factoring: GCF, difference of squares, trinomials
   - Quadratic equations (factoring, quadratic formula)

3. **Functions**
   - Domain and range
   - Function notation and composition
   - Inverse functions
   - Linear, quadratic, polynomial, rational functions

4. **Trigonometry Introduction**
   - Angle measure (degrees, radians)
   - Unit circle basics
   - Basic trig identities (Pythagorean, reciprocal)
   - Solving simple trig equations

### Problem Sets

- **Set 1B**: Polynomial factoring (15 problems)

### Topics Covered

#### Differential Calculus

1. **Limits**
   - Formal epsilon-delta definition (conceptual)
   - Limit laws and techniques
   - L'Hôpital's rule

2. **Continuity**
   - Continuous functions
   - Intermediate Value Theorem
   - Extreme Value Theorem

3. **Differentiation**
   - Definition as rate of change
   - Derivative rules: power, product, quotient, chain
   - Implicit differentiation
   - Derivatives of exponential, logarithmic, trig functions

4. **Applications**
   - Curve sketching
   - Related rates problems
   - Optimization (max/min problems)
   - Newton's method

#### Linear Algebra

1. **Vectors and Vector Spaces**
   - ℝ^n vector operations
   - Linear independence
   - Subspaces

2. **Matrices**
   - Matrix operations
   - Inverse matrices
   - Determinants and properties

3. **Linear Systems**
   - Gaussian elimination
   - Reduced row echelon form
   - Solution sets

4. **Eigenvalues and Eigenvectors**
   - Characteristic polynomial
   - Diagonalization
   - Applications

3. **Power Series**

   - Gradient and directional derivatives

2. **Multiple Integrals**
   - Double integrals
   - Triple integrals

3. **Vector Calculus (Introduction)**

### Topics Covered

#### Real Analysis

1. **Foundations**
   - Ordered field axioms
   - Supremum and infimum
   - Completeness of ℝ

2. **Sequences**
   - Convergence
   - Cauchy sequences
   - Monotone Convergence Theorem

3. **Continuity**
   - Continuous functions on ℝ
   - Intermediate Value Theorem (formal)
   - Uniform continuity

4. **Differentiation**
   - Mean Value Theorem
   - Taylor's Theorem with remainder
   - L'Hôpital's rule (formal proof)

5. **Integration**
   - Riemann integration
   - Lebesgue integration concepts (introduction)

#### Abstract Algebra

1. **Group Theory**
   - Group axioms
   - Subgroups, cyclic groups
   - Cosets and quotient groups

2. **Homomorphisms**
   - Group homomorphisms
   - Kernel and image
   - Isomorphism theorems

3. **Rings and Fields**
   - Ring axioms
   - Polynomial rings
   - Ideal theory

4. **Advanced Linear Algebra**
   - Linear transformations
   - Dual spaces
   - Jordan canonical form

5. **Galois Theory (Introduction)**
   - Field extensions
   - Splitting fields
   - Galois correspondence

## Evaluation Methods

### Weekly Quizzes (20 points each)

Each Friday: 45-minute assessment covering week's topics
- 5 definitions/concepts
- 10 short-answer problems
- 5 proof-oriented problems

### Monthly Exams (100 points)

| Exam | Topics | Format |
|------|--------|--------|
| **Month 1** | Arithmetic through pre-calc | 25 problems + 3 proofs |
| **Month 2** | Limits, derivatives, linear algebra | 25 problems + 3 proofs |
| **Month 3** | Integration, series, multivariable | 25 problems + 3 proofs |
| **Month 4** | Analysis + Abstract Algebra | 20 problems + 5 proofs |

### Problem Set Completion

| Category | Weight | Notes |
|----------|--------|-------|
| Daily problem sets | 40% | Must complete 80% |
| Weekly quizzes | 20% | Average score |

## Training Tips for Kesari

1. **Focus on understanding over speed** — Mathematical maturity takes time
2. **Practice proof writing** — Essential for Phase 4
3. **Use visual intuition** — 3Blue1Brown videos for geometric understanding
4. **Solve problems daily** — Muscles memory, builds intuition
5. **Review errors** — Every mistake is a learning opportunity
6. **Connect concepts** — See how topics build on each other
7. **Teach back** — Explain concepts as if teaching someone else
